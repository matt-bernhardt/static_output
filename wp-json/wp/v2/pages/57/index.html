Storage options<\/h2>\n

For storage and processing of your data during the active phase of research, there are several options:<\/p>\n

    \n
  • IS&T offers Athena lockers<\/a>, which provide storage for DLCs and research labs as well as courses and UROP projects; there are storage quotas.<\/li>\n
  • IS&T also offers Dropbox<\/a> accounts to all MIT students, faculty and staff, with 200 GB storage capacity and access to data using any web browser.<\/li>\n
  • For groups that need to run their own servers, IS&T maintains managed servers<\/a>\u00a0and server co-location<\/a>.<\/li>\n<\/ul>\n

    High-performance computing resources include:<\/p>\n

      \n
    • The Massachusetts Green High-Performance Computing Center<\/a> (MGHPCC) provides high-performance processing to MIT researchers.<\/li>\n
    • The Harvard-MIT Data Center Research Computing Environment (RCE)<\/a> provides storage and processing capacity; users have remote access to their data and to specialized data analysis software<\/a>.<\/li>\n<\/ul>\n

      Need help determining the right storage option for your group? IS&T’s Community Partnerships team and the Libraries’ Data Management Services can help! Contact Data Management Services<\/a> to get started.<\/p>\n

      Back up your data<\/h2>\n
        \n
      • Make 3 copies (e.g. original + external\/local + external\/remote)<\/li>\n
      • Copies should be geographically distributed (local vs. remote)<\/li>\n<\/ul>\n

        Data backup options<\/h2>\n
          \n
        • Personal computer hard drives, external hard drives, departmental or university servers<\/li>\n
        • CDs or DVDs aren’t recommended, because they fail frequently<\/li>\n
        • MIT\u2019s IS&T has free computer backup<\/span><\/a> services<\/li>\n
        • Cloud storage: several commercial options are available<\/a>; each have different requirements, encryption, and storage fees<\/li>\n<\/ul>\n

          Security<\/h2>\n
            \n
          • Unencrypted security is ideal for storing your data so that you and others can easily read it, but if encryption is required because of sensitive data:\n
              \n
            • Keep passwords and keys on paper (2 copies) and in a PGP (pretty good privacy) encrypted digital file.<\/li>\n
            • Don\u2019t rely on 3rd party encryption alone.<\/li>\n<\/ul>\n<\/li>\n
            • Uncompressed is also ideal for storage, but if you need to do so to conserve space limit compression to your 3rd backup copy.<\/li>\n<\/ul>\n

              To make sure your backup system is working properly, test your system periodically. Try to retrieve data files and make sure you can read them.<\/p>\n

              Need more help?<\/h2>\n
